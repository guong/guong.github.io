# 语法

使用方法为前后冒号包围表情代号的句法。

:blush:
1

它会在浏览器中显示为这个表情 😊 , 所以在提交代码的时候可以这样写：

git commit –m 'commit some changes :blush:'
1

一种好的表情使用规范约定：

  • 🐞 Bug 修复
  • 💄 样式更新/less 变量更新
  • 🆕 新增特性
  • 🔥 极其值得关注的新增特性
  • ✅ 新增或更新测试用例
  • 🛎 更新警告/提示信息
  • 🗑 废弃或移除
  • 🛠 重构或工具链优化
  • ⚡️ 性能提升
  • ⌨️ ♿️ 可访问性增强
  • 📖 📝 文档或网站改进
  • 🇺🇸 🇨🇳 🇬🇧 国际化改动，注意这里直接用对应国家/地区的旗帜。
